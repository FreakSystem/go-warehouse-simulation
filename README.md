# Симуляция склада (Go)

Учебный проект на Go, который показывает, как работают горутины, каналы и мьютексы в конкурентном программировании.

---

## Что делает проект

Есть склад с тремя товарами: кола, вода и фанта.

- **Поставщик** (`producer`) — каждые 4 такта добавляет случайное количество каждого товара.
- **Покупатель** (`consumer`) — каждые 3 такта пытается купить один случайный товар.
- **Инспектор** (`inspector`) — каждые 10 тактов выводит отчёт:
  - сколько товаров добавлено всего,
  - сколько успешных покупок,
  - сколько неудачных покупок (когда товара не было),
  - текущий остаток на складе.

Все три процесса работают параллельно в отдельных горутинах. Доступ к складу синхронизируется через мьютексы, а события передаются через канал.
